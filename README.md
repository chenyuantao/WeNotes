# WeNotes

## Getting Started

Start the server,

```bash
$ cd ./server
$ npm install
$ npm run start
```

Start the client,

```bash
$ cd ./client
$ npm install
$ npm run start
```

## Features

- 💪 100% Typescript (Umijs + Nestjs)
- ⏱️ Realtime (socket.io)
- 💾 Online (Mysql + Redis)
- 📒 CRUD
